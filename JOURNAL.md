# Overview
I wanted to make a system that functions like a radar basically. I watched a couple videos about it, and since i am highly interested in things like that (Some might even say, obsesssed), I decided to make this.
The idea is a sensor is hooked up to the microcontroller and displays. It scans the environment to detect whether any foreign objects have entered the space and relays that information back to the main system to be displayed. Both position and distance are accouted for and the speed of the object (if moving) can be calculated with some basic math. The sensor is also placed on a servo motor so it can have a wide angle to operate instead of just straight infront of it.
I structed the system the system into parts so i could decide what pieces of hardware could handle those parts effectively
